This is a user message. Usually it's in a P tag, but can also be a div or a span. 

<p wf-message class="compact"
    message-class="{{userMessageType}}"
    dismissable="false"
    highlight
    time-to-show="5000"
    on-dismiss="scopeFunction()"
>
    This is a user message.  Usually it's in a P tag, but can also be a div or a span.
</p>
Type of Message Message Type:
Options

Usage

  • Use the class "message" for notices to the user.
  • Then, use the class for the type of message it is.
  • The "compact" class works with messages.
  • Speak the user's language. Address him/her directly and use the word "you" if appropriate.
  • Be sure grammar and spelling are correct.
